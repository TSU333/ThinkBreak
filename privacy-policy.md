# Privacy Policy for ThinkBreak

ThinkBreak does not collect, store, or transmit any personal data.

The extension operates entirely within the user's browser and only detects page state on supported AI websites (such as ChatGPT, Claude, Gemini, and Grok) to determine whether AI is generating a response.

User settings such as the extension enable state, waiting threshold, and selected break website are stored locally using the browser's storage API. These settings never leave the user's device.

ThinkBreak does not:

- collect personal information
- track browsing history
- transmit data to external servers
- sell or share user data

If you have any questions, please contact the developer through the project repository.
